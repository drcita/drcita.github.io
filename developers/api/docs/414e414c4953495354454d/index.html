Doctor

Up 
get /medical/doctors/search
Get all medical doctor using filters (getFilteredDoctors)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request headers

Query parameters

appointment_type (required)
Query Parameter — type of appointment, 1: face-toface, 2: teleconsulting, 3: both format: hex24
city_code (optional)
Query Parameter — city code including both state and city DANE codes separated by a dash, ie 76-001
speciality_id (required)
Query Parameter — both speciality and subspeciality ids are allowed here
plan_id (optional)
Query Parameter — plan id

Return type

array[Doctor]

Example data

Content-Type: application/json
